# forever and always,
## my baby you'll be

### what is it?
this is a behavior pack for bedrock minecraft that will keep supported entities as their baby form when given the name 'baby'

### usage
* copy to `bedrock_server/behavior_packs`
* add to `bedrock_server/worlds/world_name/world_resource_packs.json`:
```
[
        {
                "pack_id": "86a220a2-5eb6-473c-b786-b8665dc02adc",
                "version": [0, 0, 1]
        }
]
```
* name a nametag `baby` and slap it on a baby mob

### supported mobs
* axolotl
* bee
* cat
* chicken
* cow
* goat
* happy ghast
* horse
* nautilus
* pig
* rabbit
* sheep
* sniffer
* tadpole (may not work?)
* turtle
* wolf
